wLeaf 3 - A ZNC management bot
===============================

Introduction
------------
wLeaf is a bot made for TreeFamily to manage our ZNC server.
The first 2 versions have been written from scratch in PHP

Requirements
------------
######RubyGems

######cinch (IRC bot framework)
	gem install cinch
######mysql-ruby
	gem install mysql
